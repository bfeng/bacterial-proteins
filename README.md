# Bacterial Protein Clustering and Visualization

## Prerequisite

1. open-mpi
2. DSC-SPIDAL algorithms

## Compile and Install

1. clone this project
2. run `mvn install`

## Run with the sample data

```bash
cd scripts
./run_all.sh
```


## Visualize

Upload the output file `damds-points.txt.formatted` to the WebPlotViz
